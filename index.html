HIỆU ỨNG HOVER

Khi lập trình một trang Website, ngoài việc nội dung mang thông tin hữu ích đến người đọc, thì hiệu ứng cũng là 1 phần giữ chân người dùng ở lại lâu hơn trang web! Một vài hiệu ứng đơn giản khi trỏ chuột đến một phần tử trong trang web là Hiệu ứng Hover.

Kết quả tham khảo

- Hover vào một đối tượng bất kì

- Hover vào toàn bố cục

Code tham khảo:

Code html

        <div class="hover-container2">
          <div class="hover-item"></div>
          <div class="hover-item"></div>
          <div class="hover-item"></div>
          <div class="hover-item"></div>
          <div class="hover-item"></div>
          <div class="hover-item"></div>
        </div>

- Hover vào một đối tượng bất kì: Trong cấu trúc code CSS này, chúng ta sử dụng CSS Pseudo Classes :not() áp dụng cho các thẻ con.

Code css


          .hover-container > .hover-item:not(:hover) {
              scale: 0.8;
              opacity: 0.7;
          }

- Hover vào toàn bố cục: Trong cấu trúc code CSS này, chúng ta sử dụng CSS Pseudo Classes :has() áp dụng cho thẻ cha.

Code css


          .hover-container2:has(:hover) > .hover-item:not(:hover) {
            scale: 0.8;
            opacity: 0.7;
          }
